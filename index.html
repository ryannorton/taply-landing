Features

Taply is built with the home studio musician in mind. Hit record, pick up the sticks, and start jamming.

Haptic Feedback

Tiny vibration motors react to each strike of the stick. It's the next best thing to playing an actual drumset.

Low Latency

We know that low latency is critical when performing live. Taply is engineered from start to finish with this in mind. All heavy computing is done directly on board the device, which means lighting fast response times, under 20ms.

Seamless Integration

Setting up Taply is a breeze. Simply connect to your computer via bluetooth and treat Taply like any other midi controller.
